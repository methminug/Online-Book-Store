### `'The world's books at your fingertips'`

## Project Description 
BookLab is an online book store where customers can buy their favourite books or add them to wishlists for the future.

## Tools and Technology used 
This website was developed with the MERN Stack

. | Technology/ Tools
--| ----------
Frontend | ReactJS
Backend | Node/Express
Database | MongoDB
Project Management |  Azure DevOps
Testing | Selenium
Version Control | GitHub
Code quality | SonarQube

## My Contributions

My contributios to this project were the end-to-end implementations of the following features: <br />
- User login and registration
- Account management
- Managing and buying from wishlists
- Managing system administrators
- Report generation (for wishlists)
